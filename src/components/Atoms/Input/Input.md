```js
  <h4>Input with prefix</h4>
  <p>(This example also ensures there are two decimal places onBlur)</p>
  <Input
    name="price"
    type="number"
    label="label"
    errorMsg=""
    id="price"
    showLabel={true}
    prefix="£"
    placeholder="0.00"
    onBlur={event => {
      const price = event.target.value;
      if (typeof Number(price) === 'number') {
        event.target.value = Number(price).toFixed(2);
      }
    }}
  />
```
